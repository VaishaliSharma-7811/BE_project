# 🚀 CryptoVista

CryptoVista is a crypto-based web application developed as a college project.  
It provides users with an interactive platform to view, manage, and simulate cryptocurrency trading activities.

---

## 📌 Project Overview

CryptoVista allows users to:

- 📊 View cryptocurrency data
- 💰 Buy and Sell crypto (simulation)
- 👤 Register and Login
- 📈 Track portfolio performance
- 🧾 View transaction history

This project is built with a scalable architecture so backend integration can be extended easily.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone <your-repo-link>
```

### 2️⃣ Navigate into the Project Folder

```bash
cd cryptoVista
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Application

For Create React App:
```bash
npm start
```

For Vite:
```bash
npm run dev
```
